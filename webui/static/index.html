🆔 CAN addresses — Cluster bus

Modifiable IDs used on the cluster CAN bus (CAN0 = MCP2515 #0).

🔍 CAN addresses — OBD2 bus

Modifiable IDs used on the OBD-II port CAN bus (CAN1 = MCP2515 #1). We TX a UDS query and wait for the engine ECU response.

📤 TX — UDS request to engine ECU

Frame sent every 200 ms: 7E0#03.22.39.C0.00.00.00.00

0x03 = ISO-TP length, 0x22 = ReadDataByIdentifier SID, 39 C0 = DID (Saugrohrdruck / MAP)

📥 RX — Expected UDS response

Expected response format: 7E8#LEN.62.39.C0.MAP_HI.MAP_LO.…

0x62 = 0x22 + 0x40 (positive response SID), then DID echo, then 16-bit MAP value big-endian (mbar)

🎯 MAP → boost gauge mapping

⚙️ Behavior

When to TX Motor_09 to cluster ?

In normal operation (force OFF), BOT32 only TXs when lever is in S/M/N. In P/R/D, BOT32 stays silent and the cluster shows real engine coolant.
